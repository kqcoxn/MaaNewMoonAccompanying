## 使用方式

当前可以使用 **MaaPiCli** 或 **MFW GUI** 运行，根据命令行提示输入相应编号即可。

功能要求：

> **目前所有功能未适配连续执行，请单个执行！**

|   功能   |   初始界面    |            注意事项            |
| :------: | :-----------: | :----------------------------: |
| 自动登录 |  进入游戏前   |        暂不支持活动奖励        |
| 一键收菜 |   首页/地图   |               -                |
| 自动爬塔 | 层数选择界面  |         需开启自动战斗         |
| 每日采购 | 首页/采购中心 | 最后会停留在流动市场数构银部分 |
